# Valentine's Day Website for Taylor üíï

A cute interactive website to ask Taylor to be your Valentine!

## Features
- Escaping "No" button that runs away from the mouse
- Big "Yes" button (3x larger than No)
- Floating hearts animation
- Confetti celebration on the success page
- Mobile-friendly with touch support

## How to Add Your Photos

1. Add two photos to this folder:
   - `photo1.jpg` - Photo for the main question page
   - `photo2.jpg` - Photo for the success/celebration page

2. The photos can be any common format (jpg, png, etc.) - just update the filenames in the HTML if different

## How to Deploy (Free!)

### Option 1: Netlify (Easiest - Drag & Drop)
1. Go to https://app.netlify.com/drop
2. Drag this entire folder onto the page
3. Done! You'll get a URL like `random-name.netlify.app`
4. (Optional) Click "Site settings" to customize the URL

### Option 2: GitHub Pages
1. Create a new GitHub repository
2. Upload all files from this folder
3. Go to Settings ‚Üí Pages
4. Select "main" branch and save
5. Your site will be at `yourusername.github.io/repo-name`

### Option 3: Vercel
1. Go to https://vercel.com
2. Sign up/login with GitHub
3. Import your repository or drag & drop files
4. Done!

## Files Included
- `index.html` - Main page with the question
- `yes.html` - Celebration page after clicking Yes
- `README.md` - This file

## Customization

To change the text, edit these files:
- In `index.html`: Change "Taylor" to any name in the `<h1>` tag
- In `yes.html`: Change "Big Dawg" or any text in the `<h1>` and `<h2>` tags
